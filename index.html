Cancer Mortalities by State 1999-2017
This map illustrates the cancer mortalites every other year from 1999-2017 in each state using proportional symbols.

How to use:

Use the slider or direction arrows to toggle between years. The legend will update the symbol size and mortalities for the year selected. You can toggle the basemap between aerial imagery and topographic, and also turn off the borders/places for more control using the icon in the upper-right portion of the map.
